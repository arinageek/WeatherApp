# WeatherApp

Погодное приложение, написанное под OC Android на языке Java с возможностью просмотра погодных условий в разных городах мира.
Имеется возможность сохранения городов и настройки уведомлений для конкретной локации в определенное пользователем время.

## Скачать

Android Studio version 4.1

compileSdkVersion: 30, minSdkVersion: 28

Склонируйте данный репозиторий и откройте его в **Android Studio**
```bash
git clone https://github.com/arinageek/WeatherApp.git
```
